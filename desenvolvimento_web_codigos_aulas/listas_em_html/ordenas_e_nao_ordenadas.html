Listas ordenadas e não ordenadas

As listas são uma maneira eficaz de organizar informações em uma página da web. Existem dois tipos principais de listas que podem ser usadas em HTML: listas ordenadas e listas não ordenadas.

Listas não ordenadas

  • Item 1
  • Item 2
  • Item 3

Uma lista não ordenada é uma lista de itens que não têm uma ordem específica. Eles são geralmente apresentados com marcadores (pontos, círculos, quadrados, etc.) que são usados ​​para destacar cada item na lista.

Listas ordenadas

  1. Item 1
  2. Item 2
  3. Item 3

Uma lista ordenada é uma lista de itens que seguem uma ordem específica. Eles são geralmente apresentados com números ou letras que indicam a posição de cada item na lista.

Em resumo, listas ordenadas e não ordenadas são uma maneira útil de organizar informações em uma página da web. Use listas não ordenadas para itens que não têm uma ordem específica e listas ordenadas para itens que seguem uma ordem específica.
